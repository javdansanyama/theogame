# AGENTS.md

## Project
Kid-friendly Phaser + Vite PWA game.

## Setup
Run these commands from the repo root:

```bash
npm install
npm run dev
npm run build
```

## Definition of Done
- The game runs well on mobile and touch devices.
- UI elements are large and easy to tap.
- No tiny text.
- No ads.
- No tracking.

## Code Style
- TypeScript is preferred for new code.
- Keep files small and readable.
- Avoid heavy dependencies unless absolutely necessary.

## Testing & Verification
Before marking work complete:
- Start the dev server successfully with `npm run dev`.
- Build the project successfully with `npm run build`.
